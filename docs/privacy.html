TaxSuite

プライバシーポリシー

最終更新日：2025年4月17日

TaxSuite（以下「本アプリ」）は、フリーランス・個人事業主の方が経費・収入を記録・管理するための iOS アプリです。本プライバシーポリシーでは、お客様の情報をどのように取り扱うかを説明します。

1. 収集する情報

端末内のみ保存

経費・収入データ

入力された経費・収入の記録（項目名、金額、日付、カテゴリ、プロジェクト、コメント）はすべてお客様の端末内（SwiftData）にのみ保存されます。当社のサーバーには送信されません。

端末内のみ保存

領収書画像

Proプランのスキャン機能で撮影した領収書のサムネイル画像は、端末内にのみ保存されます。外部サーバーへのアップロードは行いません。

オプション

Googleアカウント情報

Google連携機能（Gmailへの下書き保存・Google Driveへのエクスポート）を利用する場合、Googleアカウントのメールアドレスおよび表示名を取得します。これらは端末内にのみ保持され、当社サーバーへの送信は行いません。

利用するGoogleのスコープ：

  • gmail.compose（下書き作成・メール送信）
  • drive.file（本アプリが作成したファイルへのアクセス）
オプション・匿名

位置情報

場所トリガー機能を利用する場合、登録した地点への接近を検知するために位置情報を使用します。位置情報は端末内のみで処理され、外部には送信されません。

2. 情報の利用目的

  • 経費・収入データの記録・表示・分析
  • 領収書のスキャンおよびOCR処理
  • Google連携機能（任意）の提供
  • 場所トリガーによる自動記録（任意）
  • アプリ機能の改善（クラッシュレポート等、匿名情報のみ）

3. 第三者への提供

本アプリは、お客様の個人情報を第三者に販売・貸与・提供することは一切ありません。

ただし、以下のサービスを利用するため、各社のプライバシーポリシーが適用されます：

  • Google Sign-In / Gmail API / Google Drive API（Google LLC）
  • Google AdMob（広告表示、無料プランのみ）

4. データの保存と削除

すべての経費・収入データはお客様の端末内にのみ保存されます。アプリをアンインストールすると、端末内のデータはすべて削除されます。

iCloudバックアップが有効な場合、iOS標準のiCloudバックアップにデータが含まれることがあります。

5. 未成年者のプライバシー

本アプリは13歳未満のお子様を対象としていません。意図せず13歳未満のお子様の情報を収集した場合は、下記連絡先までご連絡ください。

6. プライバシーポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更がある場合はアプリ内またはこのページで通知します。定期的にご確認ください。
